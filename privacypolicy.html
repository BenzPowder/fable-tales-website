นโยบายความเป็นส่วนตัวนี้อธิบายถึงวิธีการทำงานของ "Fable Tales Video Generator" (ต่อไปนี้จะเรียกว่า "แอปพลิเคชัน", "เรา", "ของเรา") ซึ่งเป็นสคริปต์ Python ที่พัฒนาขึ้นเพื่อสร้างและโพสต์วิดีโอนิทานไปยังแพลตฟอร์มโซเชียลมีเดียต่างๆ

1. ข้อมูลที่เราไม่เก็บรวบรวมโดยตรง

แอปพลิเคชันของเราเป็นสคริปต์ที่ทำงานบนเครื่องคอมพิวเตอร์ของคุณเป็นหลัก โดยจะไม่เก็บรวบรวม จัดเก็บ หรือส่งข้อมูลส่วนบุคคลใดๆ จากคุณไปยังเซิร์ฟเวอร์ของเรา (ในฐานะผู้พัฒนา) โดยตรง

2. ข้อมูลที่แอปพลิเคชันประมวลผลและใช้งาน

แอปพลิเคชันนี้ประมวลผลข้อมูลที่คุณเป็นผู้ให้มา และโต้ตอบกับ API ของบุคคลที่สาม ดังนี้:

  • เนื้อหาเรื่องเล่า (Text Content): ข้อความเรื่องเล่าใดๆ (.txt files) ที่คุณป้อนให้แอปพลิเคชัน จะถูกประมวลผลภายในเครื่องของคุณเพื่อสร้างเสียงและวิดีโอ ข้อความเหล่านี้จะไม่ถูกส่งไปยังเซิร์ฟเวอร์ของเรา (ในฐานะผู้พัฒนา)
  • ไฟล์เสียง/วิดีโอ: ไฟล์เสียง (.wav) และวิดีโอ (.mp4) ที่สร้างขึ้นจะถูกจัดเก็บไว้ในเครื่องคอมพิวเตอร์ของคุณ และอาจถูกอัปโหลดโดยแอปพลิเคชันไปยังแพลตฟอร์มโซเชียลมีเดียของบุคคลที่สาม ตามการตั้งค่าของคุณ
  • ข้อมูลรับรอง API ของบุคคลที่สาม: ในการโพสต์ไปยังแพลตฟอร์มโซเชียลมีเดีย (เช่น YouTube, TikTok, Facebook) แอปพลิเคชันจำเป็นต้องใช้ข้อมูลรับรอง API ของคุณ (เช่น Google OAuth tokens, TikTok App Key/Secret, Facebook Page Access Token) ข้อมูลเหล่านี้จะถูกจัดเก็บไว้ในเครื่องของคุณ (เช่น ไฟล์ `token.json` สำหรับ YouTube) และใช้เพื่อยืนยันตัวตนกับแพลตฟอร์มเหล่านั้นเท่านั้น เรา (ในฐานะผู้พัฒนา) ไม่สามารถเข้าถึงข้อมูลรับรองเหล่านี้ได้

3. การใช้งานข้อมูล

ข้อมูล (ข้อความ, สื่อที่สร้างขึ้น) ถูกนำไปใช้เพื่อวัตถุประสงค์ในการสร้างวิดีโอและโพสต์ไปยังบัญชีโซเชียลมีเดียที่คุณกำหนดค่าไว้เท่านั้น

4. บริการของบุคคลที่สาม

แอปพลิเคชันของเรามีการเชื่อมต่อกับบริการของบุคคลที่สามดังต่อไปนี้:

  • Google Cloud Text-to-Speech: ข้อความจะถูกส่งไปยัง API ของ Google เพื่อสร้างเสียง โปรดดู ข้อกำหนดในการให้บริการของ Google Cloud และ นโยบายความเป็นส่วนตัวของ Google
  • Google Translate (ผ่าน `googletrans`): ข้อความอาจถูกส่งเพื่อแปล โปรดดู นโยบายความเป็นส่วนตัวของ Google
  • gTTS (Google Text-to-Speech): ข้อความจะถูกส่งไปยังบริการของ Google เพื่อสร้างเสียงพูด โปรดดู นโยบายความเป็นส่วนตัวของ Google
  • YouTube Data API: วิดีโอจะถูกอัปโหลดไปยังช่อง YouTube ของคุณ โปรดดู ข้อกำหนดในการให้บริการของ YouTube และ นโยบายความเป็นส่วนตัวของ Google
  • TikTok Open Platform API: ใช้สำหรับอัปโหลดวิดีโอไปยัง TikTok โปรดดู นโยบายความเป็นส่วนตัวของ TikTok
  • Facebook Graph API: ใช้สำหรับอัปโหลดวิดีโอไปยัง Facebook โปรดดู นโยบายข้อมูลของ Facebook

การใช้งานบริการของบุคคลที่สามเหล่านี้อยู่ภายใต้ข้อกำหนดและนโยบายความเป็นส่วนตัวของแต่ละบริการ เราไม่รับผิดชอบต่อการปฏิบัติด้านความเป็นส่วนตัวหรือเนื้อหาของบริการบุคคลที่สามเหล่านี้

5. ความปลอดภัยของข้อมูล

ข้อมูลที่ละเอียดอ่อนทั้งหมด (เช่น ข้อมูลรับรอง API) จะถูกจัดเก็บไว้ในเครื่องคอมพิวเตอร์ของคุณ และเรา (ในฐานะผู้พัฒนา) ไม่ได้เข้าถึงหรือจัดเก็บข้อมูลเหล่านี้ เราใช้มาตรการความปลอดภัยที่เหมาะสมเพื่อปกป้องความสมบูรณ์ของแอปพลิเคชัน แต่ไม่มีวิธีการส่งข้อมูลผ่านอินเทอร์เน็ตหรือวิธีการจัดเก็บข้อมูลอิเล็กทรอนิกส์ใดๆ ที่ปลอดภัย 100%

6. การเปลี่ยนแปลงนโยบายความเป็นส่วนตัว

เราอาจปรับปรุงนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว เราจะแจ้งให้คุณทราบเกี่ยวกับการเปลี่ยนแปลงใดๆ โดยการโพสต์นโยบายความเป็นส่วนตัวฉบับใหม่บนหน้านี้
